# OnlineBusinessValidator

A Streamlit-based web application that verifies the authenticity of any online business using:
- WHOIS lookup
- Google review scraping (no API required)
- Public review site scraping
- AI-powered authenticity scoring (optional OpenAI key)
- Automatic Excel report export

### **1. WHOIS Domain Check**
Extracts:
- Domain age
- Registrar
- Organization

### **2. Google Reviews (Mandatory Source)**
Scrapes publicly visible Google review text without needing an API.

### **3. Other Review Sources (Optional)**
Scrapes:
- Trustpilot pages
- Site review blogs
- Any publicly indexed review pages via DuckDuckGo

### **4. AI Analysis (Optional)**
Uses OpenAI API to provide:
- Legitimacy score
- Ethical assessment
- Red flags
- Final recommendation

### **5. Excel Report Export**
Generates a downloadable `validation_report.xlsx`.

### **If using GitHub + Streamlit Cloud:**
1. Create a new GitHub repository named **OnlineBusinessValidator**.

5. Connect your repository

7. Launch

